# Otter-ly Hilarious!

[API Used](https://icanhazdadjoke.com/api)

## About the app

Otter-ly Hilarious is an Android app that allows users to view dad jokes and save their favorites to view at a later date.

## Features

+ Users can create an account, or sign into an existing account.
+ Users can request dad jokes from the API via a page in the app.
+ Users can save their favorite dad jokes to the database and view them later.
+ Users can change their passwords and/or delete their account.

## Test Account

To use the app without creating an account, and to see some pre-populated data, the following account information can be used on the login page after launching the app:

+ Username: `User`
+ Password: `password`

*(both case-sensitive)*
